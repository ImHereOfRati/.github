- 출발/도착 시 자동으로 문자를 보내드립니다.
- `위치정보의 보호 및 이용 등에 관한 법률` 을 피하기 위해 서버로 실시간 위치 전송은 하지 않습니다.
- 대신, `Mobile` 에서 `GPS` 을 지속적으로 추적하고, 조건이 일치할 때, 알림 발송 요청을 서버로 보냅니다.
